# 🎯 Example

From **Reverse Linked List II**

```
1 → 2 → 3 → 4 → 5
left = 2, right = 4
```

# 🧩 STEP 0: Initial Setup

```
dummy → 1 → 2 → 3 → 4 → 5
          ↑
        prev
```

After moving `prev`:

- `prev = 1`
- `curr = 2`

---

# 🔄 STEP 1: Start Reversing

### 👉 Before change:

```
prev → 1
curr → 2 → 3 → 4 → 5
subPrev → null
```

---

### 🔁 Iteration 1

```java
next = 3
curr.next = null
subPrev = 2
curr = 3
```

### 👉 State now:

```
Reversed part: 2 → null

Remaining:
1 → 2    3 → 4 → 5
        ↑
      curr
```

---

# 🔄 Iteration 2

```java
next = 4
curr.next = 2
subPrev = 3
curr = 4
```

### 👉 State now:

```
Reversed part: 3 → 2 → null

Remaining:
1 → 2    4 → 5
        ↑
      curr
```

---

# 🔄 Iteration 3

```java
next = 5
curr.next = 3
subPrev = 4
curr = 5
```

### 👉 State now:

```
Reversed part: 4 → 3 → 2 → null

Remaining:
1 → 2    5
        ↑
      curr
```

---

# 🧠 STOP! Key Observation

Now we have:

- `subPrev` → head of reversed list (`4`)
- `tail` → original `2` (now last)
- `curr` → `5` (rest of list)

---

# 🔗 STEP 2: Reconnect

### Before reconnect:

```
1 → 2     4 → 3 → 2     5
    ↑       ↑
  prev   subPrev
```

---

### 🔗 Connect front:

```java
prev.next = subPrev;
```

```
1 → 4 → 3 → 2
```

---

### 🔗 Connect back:

```java
tail.next = curr;
```

```
1 → 4 → 3 → 2 → 5
```

---

# ✅ FINAL OUTPUT

```
1 → 4 → 3 → 2 → 5
```

# ⚡ Pointer Roles (REMEMBER THIS)

| Pointer   | Meaning              |
| --------- | -------------------- |
| `prev`    | before reversal      |
| `curr`    | node being processed |
| `next`    | backup               |
| `subPrev` | reversed head        |
| `tail`    | end of reversed part |
